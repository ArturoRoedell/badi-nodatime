About the Calendar

The Wondrous calendar (also known by its name in Arabic: Badí‘) was introduced by the founders of the Bahá'í Faith, the Báb and Bahá'u'lláh, in the mid- to late-1800's (in the Gregorian calendar). More details can be found at the Wikipedia page about the Wondrous Calendar.

Installation

You can install this via nuget at NuGet: Wondrous Noda Time .

Usage

Using

{% highlight c# %}using WondrousNodaTime;{% endhighlight %}

Basic Dates

WondrousNodaTime has complete support for translating dates between the Wondrous calendar and the Gregorian calendar. Internally, Noda Time LocalDates with the `CalendarSystem.Wondrous` are used. If you use LocalDates, the full power of Noda Time is available to you. If you are not using Noda Time, you don't need to know anything about it!

{% highlight c# %} // show today's date var today = new WondrousDate(); Console.WriteLine("Today is " + today.ToString()); // sample result: Today is 174-15-9 // convert a Wondrous date to a Gregorian date var gregorianDate = today.DateTime; // create a WondrousDate from a .NET DateTime var date1 = new WondrousDate(DateTime.Today); // create a WondrousDate from a Noda Time LocalDate var localDate = new LocalDate(DateTime.Today); var date2 = new WondrousDate(localDate); // create a WondrousDate for a specific date var date3 = new WondrousDate(175, 1, 1); // Naw Ruz 175 // subtract one day then get the Gregorian date var gregorianDate2 = date3.PlusDays(-1).DateTime; Console.WriteLine("The day before Naw Rúz 175 is " + gregorianDate2.ToString("D")) // sample result: The day before Naw Rúz 175 is Tuesday, March 20, 2018 {% endhighlight %}

Bahá'í Holy Days

This version of the library has full knowledge of the Bahá'í Holy Days and what days they fall on.

{% highlight c# %} var year174 = new WondrousYearInfo(174); var listing = year174.GetSpecialDays(SpecialDayType.HolyDay_WorkSuspended | SpecialDayType.FeastDay); // more samples to come... {% endhighlight %}

Date Formatting

The `ToString` method of each `WondrousDate` object supports a non-standard extended formatting system. The `patternText` you supply is actually a template with a set of custom tokens that will be processed using that date.

{% highlight c# %} var pattern = "{day} {month_arabic} {yearOfEra}"; Console.Writeline(new WondrousDate(180, 10, 9).ToString(pattern)); // result: 9 `Izzat 180 pattern = "{day_meaning} {month_meaning} {yearOfUnity}"; // result: Names Might 9 pattern = "{weekday_arabic}/{weekday_meaning} ({element_meaning}) in the {yearOfUnity_ordinal} year of the {unity_ordinal} Unity."; // result: Jalál/Glory (Water) in the nineth year of the tenth Unity. {% endhighlight %}

The names used can be extended into your spoken language quite easily. Languages already in the package include: English, French, German, Dutch, Spanish, Portugese, Chinese, Esperanto, and Tamil. (Some are only partially translated.) For example, in German:

{day_meaning}
Wille
{month_meaning}
Fragen
