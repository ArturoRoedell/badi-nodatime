This library provides comprehensive support for the Wondrous calendar in .NET projects.

About the Calendar

The Wondrous calendar (also known as the Badí‘ calendar) was introduced by the founders of the Bahá'í Faith, the Báb and Bahá'u'lláh, in the mid- to late-1800's (according to the Gregorian calendar). More details can be found at the Wikipedia page about the Wondrous Calendar.

Installation

You (will be able to) install this via nuget.
